	System for saving sales` information.
    Given: Product(name, price), Sale(Product, Quantity, Cost, Date);
    Task: need to write a site with Spring MVC with 2 pages:
    1. Products info. Page must contain product list with ability to edit existing or to add new position. Every product must have information about its sales.
    2. Sales info. Page must contain sale list with ability to add new position. Every sale must have information about its products.
    Restrictions:
	Application server: Apache Tomcat,
	Database: PostgreSQL,
	Data saving: JPA(Hibernate).

	Система хранения информации о продажах.
    Дано: Товар (Наименование, Цена), Продажа (Товар, Количество, Стоимость, Дата);
    Задание: Необходимо написать сайт с использованием Spring MVC содержащий 2 страницы:
    1. Информация о товарах. Страница должна содержать список товаров с возможностью редактирования и добавления новых позиций. По каждому товару должна присутствовать информация о его продажах.
    2. Информация о продажах. Страница должна содержать список продаж с возможностью добавления новых позиций. По каждой продаже должна присутствовать информация о товарах.
    Ограничения:
    Сервер приложений: Apache Tomcat
    База данных: PostgreSQL
    Хранение данных: JPA (Hibernate)
